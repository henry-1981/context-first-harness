# Core Inventory

## Included in Core

- `GET /slides/:file`
- `GET /api/slides`
- `POST /api/slides/:file/save`
- `GET /api/events`
- slide file ordering
- SSE broadcast on HTML file changes
- editor shell route (`/`) and static JS mount when paths are provided

## Explicitly Not in Core

- AI edit endpoint
- direct text editing controls
- object selection / bbox logic
- entity variableization
- drag / resize / delete tooling
